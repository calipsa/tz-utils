## Installation
```bash
# using npm:
npm install --save @calipsa/tz-utils

# or if you like yarn:
yarn add @calipsa/tz-utils
```

## Usage
```javascript
import {
  formatDateTz,
  names,
  offset,
  offsetFormattedLong,
  offsetFormattedShort,
  suggested,
} from '@calipsa/tz-utils'

// ...
```
